# CS 330 Final Project – 3D Scene

This project demonstrates the creation of a 3D scene using OpenGL, focusing on object composition, transformations, lighting, and textures. The final scene represents a Christmas-themed environment that includes a tree, ornaments, and a textured ground plane.

## How do I approach designing software?

When designing this project, I focused on breaking the scene into smaller components and building each piece step by step. I treated each object, such as the tree, ornaments, and ground, as separate elements that could be created and adjusted individually. 
This approach made the overall design more manageable and allowed me to make changes without impacting the entire scene.

## How do I approach developing programs?

My development process involved testing frequently and making incremental changes. I would implement one feature at a time, such as lighting or object placement, and then run the program to confirm it worked before moving on. 
When issues occurred, I used trial and error along with debugging to identify and fix problems. This helped me understand how different parts of the program interact.

## How can computer science help me in reaching my goals?

Computer science skills gained from this project will support my future goals by strengthening my ability to problem solve and build structured applications. 
Understanding how graphics rendering works also provides a foundation for more advanced areas such as game development or visualization tools. These skills will be valuable as I continue working toward a career in software development.

## What new skills have I developed?

Through this project, I developed skills in working with OpenGL, including rendering 3D objects, applying transformations, and implementing lighting effects. 
I also improved my ability to organize code using multiple classes and files, which made the project easier to manage and expand.
